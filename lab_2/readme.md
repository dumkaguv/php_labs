# Лабораторная работа №2. Установка и первая программа на PHP

## Ход работы

### Шаг 1. Установка PHP (ручная установка)

1. Перейти на официальный сайт PHP: <https://www.php.net/downloads>
2. Скачать актуальную версию PHP для своей операционной системы.
3. Распаковать архив в удобное место, например:
   `C:\Program Files\php`
4. Добавить путь к PHP в переменные среды (Path):
   - Win + R → `sysdm.cpl`
   - Вкладка **Дополнительно**
   - **Переменные среды**
   - В разделе **Системные переменные** выбрать `Path`
   - Добавить путь `C:\Program Files\php`
5. Проверить установку командой:

```bash
php -v
```

Если отображается версия PHP - установка выполнена успешно.

## Шаг 5. Работа с переменными

```php
<?php

echo "Привет, мир!";

echo "Hello, World with echo!";
print "Hello, World with print!";

$days = 288;
$message = "Все возвращаются на работу!";

echo "Количество дней: " . $days . "<br />";
echo "Сообщение: " . $message . "<br /><br />";

echo "Количество дней: $days<br />";
echo "Сообщение: $message<br />";
?>
```

### Использованные способы вывода

- Конкатенация через оператор `.`
- Вывод через двойные кавычки
- Перенос строки с помощью тега `<br />`

## Чем отличается оператор echo от print?

## Отличие `echo` и `print` в PHP

| echo                              | print               |
| --------------------------------- | ------------------- |
| Может выводить несколько значений | Выводит только одно |
| Работает быстрее                  | Чуть медленнее      |
| Не возвращает значение            | Возвращает `1`      |
| Используется чаще                 | Используется реже   |
