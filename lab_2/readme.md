# Контрольные вопросы

## Какие способы установки PHP существуют?

### Основные варианты

- Установка вручную:
  - Скачать PHP с официального сайта

  - Настроить переменные окружения

- Готовые сборки
  - XAMPP

  - OpenServer

## Как проверить, что PHP установлен и работает?

### В терминале

```bash
php -v
```

## Чем отличается оператор echo от print?

## Отличие `echo` и `print` в PHP

| echo                              | print               |
| --------------------------------- | ------------------- |
| Может выводить несколько значений | Выводит только одно |
| Работает быстрее                  | Чуть медленнее      |
| Не возвращает значение            | Возвращает `1`      |
| Используется чаще                 | Используется реже   |
