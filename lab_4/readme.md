# Лабораторная работа №4. Массивы и функции в PHP. Работа с файловой системой (галерея)

## 1. Цель работы

Освоить работу с массивами в PHP, применяя операции: создание, добавление, удаление, сортировка и поиск.  
Закрепить навыки работы с функциями: аргументы, возвращаемые значения, типизация, анонимные функции.  
Освоить работу с файловой системой: чтение содержимого директории и вывод изображений в виде галереи.

## 2. Постановка задачи

### Задание 1. Работа с массивами (транзакции)

Разработать систему управления банковскими транзакциями с возможностью:

- добавления новых транзакций;
- удаления транзакций;
- сортировки транзакций по дате или сумме;
- поиска транзакций по описанию.

Дополнительно:

- вывести список транзакций в HTML-таблице;
- вывести количество дней с даты транзакции до текущего дня;
- посчитать общую сумму всех транзакций.

### Задание 2. Работа с файловой системой (галерея)

- Создать директорию `image/` и поместить в неё 20–30 изображений `.jpg/.jpeg`.
- Считать файлы из директории и вывести их на страницу в виде галереи.

## 3. Описание решения (реализация)

### 3.1. Подготовка среды и строгая типизация

В начале файла включена строгая типизация:

- `declare(strict_types = 1);`  
  Это повышает надёжность кода и делает приведение типов более строгим.

---

### 3.2. Структура данных транзакций

Используется массив `$transactions`, где каждая транзакция — ассоциативный массив со следующими полями:

- `id` — уникальный идентификатор
- `date` — дата в формате `YYYY-MM-DD`
- `amount` — сумма (float)
- `description` — описание платежа
- `merchant` — получатель

---

### 3.3. Реализованные функции

#### `calculateTotalAmount(array $transactions): float`

Считает общую сумму всех транзакций через `array_reduce`.

#### `findTransactionByDescription(array $transactions, string $descriptionPart): array`

Ищет транзакции по части строки в описании (регистронезависимо).  
Реализация через `array_filter` + `stripos`.

#### `findTransactionById(array $transactions, int $id)`

Ищет транзакцию по ID.  
**Важно:** в коде используется `array_find`, которого нет в стандартной библиотеке PHP 8.  
Чтобы функция работала в PHP 8+, нужно:

- либо реализовать поиск через `foreach`,
- либо заменить на `array_filter` + выбор первого элемента.

#### `daysSinceTransaction(string $date): int`

Возвращает количество дней между датой транзакции и текущей датой, используя `DateTime` и `diff()`.

#### `addTransaction(array &$transactions, int $id, string $date, float $amount, string $description, string $merchant): void`

Добавляет транзакцию в массив (передача по ссылке).

#### `deleteTransactionById(array &$transactions, int $id): bool`

Удаляет транзакцию по ID через фильтрацию массива. Возвращает `true`, если удаление произошло.

#### `sortTransactionsByDate(array $transactions): array`

Сортирует транзакции по дате по возрастанию, используя `usort()` и `DateTime`.

#### `sortTransactionsByAmountDesc(array $transactions): array`

Сортирует транзакции по сумме по убыванию, используя `usort()` и spaceship-оператор.

#### `nextTransactionId(array $transactions): int`

Возвращает следующий уникальный ID (максимальный `id` + 1).

## 6. Контрольные вопросы

### 6.1. Что такое массивы в PHP?

Массив в PHP — структура данных, позволяющая хранить набор значений.  
Массивы бывают индексными и ассоциативными (ключ → значение).

### 6.2. Как создать массив в PHP?

Основные варианты:

- короткий синтаксис: `$a = [1, 2, 3];`
- ассоциативный: `$a = ["name" => "Ivan", "age" => 20];`

### 6.3. Для чего используется цикл `foreach`?

`foreach` используется для удобного перебора элементов массива (значений и/или ключей) без явного управления индексами.
