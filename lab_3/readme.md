## Часть 2. Циклы

### Цикл `for`

```php
$a = 0;
$b = 0;

for ($i = 0; $i <= 5; $i++) {
    $a += 10;
    $b += 5;

    echo "Step $i: a = $a, b = $b<br>";
}

echo "(For) End of the loop: a = $a, b = $b";
```

---

## Цикл `while`

```php
$a = 0;
$b = 0;

$i = 0;
while ($i <= 5) {
    $a += 10;
    $b += 5;

    echo "Step $i: a = $a, b = $b<br>";

    $i++;
}

echo "(While) End of the loop: a = $a, b = $b";
```

---

## Цикл `do-while`

```php
$a = 0;
$b = 0;

$i = 0;
do {
    $a += 10;
    $b += 5;

    echo "Step $i: a = $a, b = $b<br>";

    $i++;
} while ($i <= 5);

echo "(Do-while) End of the loop: a = $a, b = $b";
```

## Контрольные вопросы

## 1. Разница между циклами

- **for** - когда известно количество повторений
- **while** - когда количество повторений заранее неизвестно
- **do-while** - выполняется минимум один раз

---

## 2. Тернарный оператор

```php
(условие) ? значение_если_true : значение_если_false;
```

Краткая форма `if-else`.

---

## 3. Что будет, если условие в do-while ложно изначально?

Тело цикла выполнится один раз, затем цикл завершится.
